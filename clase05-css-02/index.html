Tipos de fuente

  • Serif: Poseen terminaciones con serifa o acabado elegante.
  • Sans Serif: Tipografías de palo seco con terminaciones rectas o ligeramente redondeadas.
  • Display: Normalmente utilizadas para títulos por su capacidad de resaltar.
  • Mono: Cada letra mide lo mismo de ancho, suelen ser las de máquina de escribir o terminal de computadora.
  • Handwriting: Simulan la escritura hecha a mano.
